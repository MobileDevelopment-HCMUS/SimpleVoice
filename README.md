# Simple Voice

Simple Voice is an android app which was developed for:
- Recording standard.
- Speech to text from an available record.
- Speech to text directly.
- Edit record (cut, rename, show record's details)
- Custom tone
- Repeat directly after a talk like 'My talking tom'
